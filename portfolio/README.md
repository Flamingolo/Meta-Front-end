# Portfolio

Welcome to my Portfolio Project! This project showcases my skills and projects using a React-based application.

## Features

- Interactive UI
- Responsive design
- Dynamic content

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- [Git](https://git-scm.com/) installed on your local machine.
- [Node.js](https://nodejs.org/) installed.


### Installation

1. **Clone the repository**

    ```sh
    git clone https://github.com/Flamingolo/Meta-Front-end/tree/main/portfolio
    ```
    Locate and move into the repository folder.

2. **Install dependencies**

    ```sh
    npm install
    ```

### Running the Project

To start the development server, run:

```sh
npm start
```

Open your browser to `http://localhost:3000`
